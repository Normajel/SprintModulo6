Resumen del Proyecto:
El Sistema de Gestión de Prevención de Riesgos es una aplicación web diseñada para ayudar a las empresas a gestionar y mitigar los riesgos laborales en el lugar de trabajo. 
Permite a los usuarios administrar capacitaciones, registrar visitas, gestionar pagos y mantener un seguimiento de los controles de seguridad. 
El sistema incluye funcionalidades para administrar usuarios con diferentes roles y permisos.

Aspectos Relevantes:
- El proyecto utiliza Java como lenguaje de programación y Spring Framework para la capa de negocio y acceso a la base de datos.
- Se utiliza JDBC Template para conectarse a la base de datos MySQL.
- Todas las páginas del sistema son responsivas y comparten la misma hoja de estilos CSS para garantizar un diseño coherente en todos los dispositivos.
- Se han implementado múltiples archivos JavaScript debidamente comentados para proporcionar funcionalidades interactivas en la interfaz de usuario.
- Se ha utilizado Spring Security para el manejo de sesiones de usuario y la seguridad de la aplicación.
